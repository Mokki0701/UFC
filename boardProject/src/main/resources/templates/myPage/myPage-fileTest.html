          [form태그의 enctype 속성]

          enctype 이란??
          - 데이터를 서버로 전송할 때 데이터의 형태(인코딩)를 지정하는 속성


          * application/x-www-form-urlencoded (기본값)
           - URL 인코딩된 "문자열" 로 서버에 제출


          * multipart/form-data  (무조건 POST에서만 동작)

           - 제출되는 여러 데이터 타입에 맞춰서 인코딩하여 제출
            문자열 -> String
            숫자   -> String -> int/double(Spring이 처리)
            파일   -> 2진 데이터

          * text/plain
           - 텍스트로 서버에 제출
